RUBT — Стабильный токен

Что такое RUBT?

RUBT — это стандартный ERC20 токен в сети Ethereum, который находится по адресу 0x5F3011a4...8830606D.

За 1 ETH вы всегда получите то же количество RUBT чтобы вы получили в наличных рублях и наоборот. Это главное и самое важное свойство стабильного RUBT токена.

В контракте RUBT нет и никогда не будет реализован механизм блокировок или заморозки.

Самый простой способ начать использовать RUBT — это добавить RUBT токен в кошелек. Далее обменять ETH на RUBT или наоборот.

Основные функции контракта

rates

Возвращает текущий курс обмена.

  • buyRate — курс обмена 1 ETH на RUBT
  • sellRate — курс обмена RUBT на 1 ETH

buy

Обмен ETH на RUBT. При вызове этого метода вы должны отправить ETH, который будет обменен на RUBT по текущему курсу. Обязательный параметр expectedRate — ожидаемый курс обмена, он должен быть не ниже чем установленный курс, иначе транзакция будет отменена. Результатом будет зачисление RUBT на адрес вызывающего.

  • expectedRate — ожидаемые курс обмена 1 ETH на RUBT

sell

Обмен RUBT на ETH. При вызове этого метода вы должны указать количество RUBT, который будет обменен на ETH по текущему курсу. Обязательный параметр expectedRate — ожидаемый курс обмена, он должен быть не выше чем установленный курс, иначе транзакция будет отменена. Результатом будет перевод ETH на адрес вызывающего.

  • value — количество RUBT для обмена
  • expectedRate — ожидаемые курс обмена RUBT на 1 ETH
